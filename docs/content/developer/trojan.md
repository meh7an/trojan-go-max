Trojan-Go follows the original Trojan protocol. See the [Trojan documentation](https://trojan-gfw.github.io/trojan/protocol) for the full format specification.

By default, the Trojan protocol is carried over TLS:

| Layer   |
| ------- |
| Payload |
| Trojan  |
| TLS     |
| TCP     |
