# Trojan-Go

This is the documentation for Trojan-Go. You can find usage tips and the complete configuration reference in the left navigation bar.

Trojan-Go is a full Trojan proxy implementation written in Go. It is compatible with the Trojan protocol and the original configuration file format, and supports the vast majority of Trojan-GFW features while extending many additional practical capabilities.

The primary goal of Trojan-Go is to guarantee transmission security and undetectability. Within that constraint, it aims to maximize throughput performance and ease of use.

If you encounter configuration or usage problems, discover a bug, or have ideas for improvement, you are welcome to join the Trojan-Go [Telegram discussion group](https://t.me/trojan_go_chat).
